Splash Page

Overview

The Splash Page is an introductory page that sets the tone for the ClearWave initiative. It features a full-screen oceanic background, a preloader animation, and a mission statement, creating an engaging and visually appealing first impression. After 7 seconds, users are redirected to home page.

Technical Description

The Splash Page is built using HTML, CSS, and a meta refresh tag for automatic redirection. The preloader animation improves user experience, while the mission statement conveys the purpose of the ClearWave project.

Key Features

  • A preloader animation (loading-area) that visually signals the page loading process before revealing content.
  • A mission statement section, placed in the center of the screen, with a semi-transparent background to give a floating effect.
  • A team members’ section, positioned at the bottom, providing contributor credits in a structured layout.
  • The ClearWave title, enhanced with a distinctive Rubik Doodle Shadow font for strong branding.
  • A meta refresh tag that automatically redirects users to an external deep-sea exploration page.

Accessibility

  • Semantic HTML elements (<h1>, <h3>, <p>) ensure proper content structuring for screen readers.
  • Optimized text contrast enhances readability, ensuring accessibility for visually impaired users.
  • Controlled animation speeds reduce abrupt transitions, minimizing discomfort for users sensitive to motion effects.

Volunteer Page

Overview

The Volunteer Page is an interactive page showcasing volunteering opportunities for ocean conservation. It features volunteer programs, project details, and a review section, allowing users to engage with the ClearWave initiative.

Technical Description

This page is styled using css and includes interactive volunteer program cards, a filter option, and a testimonials section.

Key features

  • A responsive navigation menu allowing easy access to different website sections.
  • A hero section with an engaging call-to-action (Join Us in Fighting Sea Pollution!), visually supported by a background image.
  • A project section, formatted as a grid-based layout, showcasing multiple volunteer opportunities with:
    • High-quality images for visual appeal.
    • Project descriptions detailing the role, goals, and impact of each volunteer opportunity.
    • Eligibility criteria (age requirements, duration, fees).
    • Call-to-action buttons (Apply and View Project), which change color when hovered over, improving usability.
  • A filter system that allows users to sort projects based on popularity, recency, and rating.
  • A volunteer review section, where past participants share testimonials and rate their experiences.

Accessibility

  • High color contrast between text and background ensures readability for all users.
  • Logical heading structure (<h1>, <h3>, <h5>) for improved screen reader navigation.
  • Hover effects and clickable elements are optimized for both mouse and keyboard users.

Content Page

Overview

The Content Page serves as an informative hub to educate users about ocean conservation and encourage participation in environmental efforts. It features structured sections, each focusing on different aspects of marine conservation, such as the importance of protecting the ocean, ClearWave’s mission, ongoing projects, and ways for users to contribute.

Technical Description

The page is built using HTML and CSS, ensuring a visually appealing deep-sea theme. The content is divided into multiple interactive sections that present key environmental issues, ClearWave’s impact, and volunteer opportunities.

Key features

  • A sidebar navigation menu for quick access to different sections of the content page.
  • A hero section displaying an inspirational quote to motivate action.
  • Multiple informative sections, including:
    • Why Should We Care? → Highlights the dangers of ocean pollution.
    • Our Mission → Explains ClearWave’s key initiatives.
    • Our Impact → Displays statistics on conservation efforts.
    • Be the Wave of Change → Encourages user participation in environmental projects.
    • Our Partners → Showcases organizations collaborating with ClearWave.
    • Are You Ready? → Provides a call-to-action for joining the initiative.
  • Visually engaging images that complement the text and provide a deeper understanding of conservation efforts.
  • A structured card-based layout for ClearWave’s initiatives, making it easier for users to explore different projects.
  • A call-to-action button ("Join Now"), which directs users to the volunteer sign-up page.
  • A footer section containing useful links, social media connections, and a newsletter subscription form

Accessibility

  • Well-structured headings (<h1>,<h2>,<h3>) for clear navigation.
  • High contrast text for readability against dark backgrounds.
  • Keyboard-friendly sidebar navigation.
  • Screen reader compatibility through semantic HTML elements.

Compliance

Accessibility & Usability

The pages follow accessibility guidelines by using **semantic HTML**, **alt text for images**, **keyboard navigability**, and **contrast adjustments** to improve readability for all users.

Data Privacy & Security

No user data is stored—**all collected information is purely for display purposes**. Forms include validation to ensure proper input formatting, but **no personal data is saved** or processed beyond the session.

Content & Copyright Compliance

All images used on the website are either **royalty-free** or properly referenced. Content adheres to ethical standards, avoiding plagiarism or misinformation.

Security & Network Policies

The pages load over **secure HTTPS connections**, and external resources (fonts, icons, scripts) are sourced from **trusted CDNs** to prevent vulnerabilities.

Ethical & Legal Considerations

Our project aligns with **Janet's web publishing regulations**, ensuring responsible content management and compliance with ethical web practices.
